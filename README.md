

 Mini Dashboard / Task Manager


This is a  mini dashboard app Task Manager, Weather App  and News App**.  

It demonstrates:  
- Task management (CRUD + filters)  
- Weather API integration (OpenWeatherMap)  
- News API integration (NewsAPI for Tesla news)  
- Async programming patterns: callbacks, promises, async/await  
- Functional programming: map, filter, reduce  
- LocalStorage persistence for tasks  
- TypeScript usage for task management  



Task Manager
- Add, edit, and delete tasks  
- Mark tasks complete/incomplete  
- Filter tasks: All / Completed / Pending  
- Task statistics: Total, Completed, Pending  
- Tasks saved in localStorage to persist across sessions  

Weather App
- Search for a city to see current weather  
- Displays temperature, feels like, humidity, description, and icon  
- Handles invalid cities with error messages  

News App
- Fetches latest Tesla news using NewsAPI  
- Displays top 5 articles with title, description, and link  
- Demonstrates callback, promise, and async/await usage
